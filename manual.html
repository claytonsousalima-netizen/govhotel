Sistema de Governança Hoteleira · Todos os perfis de acesso · Versão 2026.06.3
📋 Visão Geral do Sistema

O GovHotel é um sistema web de governança hoteleira que centraliza o controle de limpeza de apartamentos, chamados internos de governança e manutenção, indicadores operacionais em tempo real e gestão da equipe — tudo integrado ao banco de dados Supabase.

Acesso: Abra o sistema no navegador e faça login com seu usuário e senha fornecidos pelo administrador. O sistema abre automaticamente na tela correta para o seu perfil. Em celular, o menu lateral é acessado pelo botão ☰ no canto superior esquerdo.

Admin HotelControla um hotel específicoTodas exceto Hotéis
GestorGestão operacional do hotelDashboard, Mapa, Kanban, Chamados, Equipe, Relatórios, Minha Fila
SupervisoraSupervisão e conferência de limpezaMapa, Kanban, Chamados, Equipe, Minha Fila
CamareiraExecução da limpezaApp Camareira, Mapa, Chamados, Minha Fila
ManutençãoManutenção de apartamentosApp Manutenção, Mapa, Chamados
Dados sempre do banco: Todos os dados (apartamentos, equipe, chamados, categorias) são carregados diretamente do Supabase. Não há dados fictícios ou de exemplo no sistema.

Prioridades:

Baixa Normal Alta Urgente

Chamados sem responsável — 📋 Disponível

Um chamado aberto sem responsável atribuído fica visível para toda a equipe do departamento correspondente no hotel. Ele é marcado com o badge Disponível na listagem e no detalhe. Qualquer membro autorizado pode clicar em ✋ Assumir chamado para se tornar o responsável.

Tipos de Chamado:

Os tipos de chamado são carregados do banco de dados por departamento (Governança ou Manutenção) e podem ser configurados pelo Admin em Config → Tipos de chamado.

🔔 Notificações em Tempo Real Atualizado
Supabase Realtime + detecção automática: O sistema usa WebSocket para notificar a equipe em tempo real e também detecta novos chamados a cada atualização do cache, garantindo que nenhuma notificação seja perdida mesmo em caso de instabilidade de conexão.
  • 1
    Quando um novo chamado é aberto, os destinatários do departamento correspondente recebem um aviso modal na tela com o número do chamado, departamento, tipo de chamado e prioridade. O aviso não fecha automaticamente — exige ação explícita.
  • 2
    O destinatário deve pressionar o botão ✓ OK — Confirmar recebimento. Ao confirmar, o sistema registra no histórico do chamado: "Recebimento confirmado por [nome] em [data/hora]".
  • 3
    Se chegarem múltiplos chamados enquanto o aviso está aberto, eles são enfileirados — ao confirmar um, o próximo aparece automaticamente.
  • 4
    A lista de chamados e o badge numérico do menu atualizam automaticamente em paralelo, sem ação do usuário — inclusive quando a camareira ou manutenção está em outras telas (Minha Fila, Meus Aptos).
  • 5
    Camareira recebe somente notificações de chamados de Governança. Manutenção recebe somente notificações de chamados de Manutenção. Demais perfis (gestor, admin, supervisora) não recebem o aviso modal, mas o badge do menu é atualizado para todos.
  • 6
    Quem abriu o chamado não recebe notificação de si mesmo.
  • 7
    A conexão é encerrada automaticamente ao fazer logout. Os chamados conhecidos são resetados a cada novo login.

Registro de Recebimento no Histórico

Após o OK, o detalhe do chamado (aba Histórico) exibirá o evento de recebimento com nome e horário do confirmante — servindo como comprovante de ciência do chamado.

Administrador Global

Controla todos os hotéis da rede. Acesso irrestrito a todas as funcionalidades.

🏨 Hotéis 👤 Usuários 📊 Dashboard 🗺 Mapa 📋 Kanban 🎫 Chamados 👥 Equipe 🛏 Cadastro Aptos 📈 Relatórios ⚙ Config 📌 Minha Fila
🏨 Gerenciar Hotéis
  • 1
    No menu lateral, clique em Hotéis.
  • 2
    Para adicionar: clique em + Novo Hotel, preencha nome, cidade, CNPJ e clique Salvar.
  • 3
    Para editar: clique no nome do hotel na lista e altere os dados.
  • 4
    Para desativar: toggle "Ativo" no cadastro. O hotel desaparece dos filtros, mas os dados são preservados.
👤 Gerenciar Usuários
  • 1
    No menu lateral, clique em Usuários.
  • 2
    Para criar: clique em + Novo Usuário, preencha nome, login (username ou e-mail), perfil, hotel vinculado e senha temporária.
  • 3
    Marque "Forçar troca de senha no primeiro acesso" — o usuário definirá nova senha ao entrar.
  • 4
    Para desativar: clique no usuário → toggle "Ativo" → Salvar. Acesso é revogado imediatamente.
  • 5
    Para redefinir senha: clique no usuário → Redefinir senha → informe a nova senha temporária.
  • 6
    Somente o Admin Global pode criar, ver e editar outros usuários com perfil Admin Global.
Isolamento por hotel: Cada usuário só enxerga dados do hotel ao qual está vinculado. O Admin Global enxerga todos os hotéis e pode selecionar o hotel desejado em cada módulo.
📊 Dashboard — Selecionar Hotel
  • 1
    No Dashboard, use o seletor 🏨 Hotel no topo para escolher o hotel.
  • 2
    Os indicadores carregam automaticamente: totais de apartamentos, status, chamados abertos, equipe ativa.
  • 3
    O mesmo seletor existe em Mapa, Kanban, Chamados, Equipe e Minha Fila.
  • 4
    Clique em ↺ Atualizar a qualquer momento para recarregar os dados.
🛏 Cadastro de Apartamentos
Exclusivo para Admin: Apenas Admin Global e Admin Hotel podem cadastrar, editar e excluir apartamentos. Os demais perfis conseguem alterar somente o status operacional.
  • 1
    Clique em Cadastro Apto no menu lateral.
  • 2
    Para adicionar: clique em + Novo Apartamento, preencha número, andar, tipo e clique Salvar.
  • 3
    Para gerar em lote: clique em ⚡ Gerar em lote e configure o padrão de numeração.
  • 4
    Para editar: clique no apartamento na tabela e altere os dados cadastrais.
  • 5
    Para excluir: clique no ícone de exclusão no card do apartamento e confirme.
⚙ Configurações do Sistema
  • 1
    Clique em Config no menu lateral.
  • 2
    Configure tipos de chamados por departamento (Governança / Manutenção). Esses tipos aparecem no formulário de novo chamado.
  • 3
    Configure tipos de solicitante (Recepção, Hóspede, Gerência etc.) — veja detalhes abaixo.
  • 4
    Configure checklists de limpeza padrão que a camareira confere ao concluir.
  • 5
    Configure tipos de apartamento (Standard, Luxo, Suíte, etc.).

Administrador do Hotel

Controla um único hotel. Mesmo acesso do Admin Global, exceto gerenciamento da rede de hotéis.

👤 Usuários 📊 Dashboard 🗺 Mapa 📋 Kanban 🎫 Chamados 👥 Equipe 🛏 Cadastro Aptos 📈 Relatórios ⚙ Config 📌 Minha Fila
O Admin Hotel tem as mesmas funções do Admin Global, porém limitadas ao seu hotel. Consulte as seções de cada módulo para instruções detalhadas. Diferenças:
  • Não aparece o menu Hotéis (gerenciamento da rede)
  • Não consegue visualizar ou criar usuários com perfil Admin Global
  • Vê automaticamente os dados do seu hotel (sem seletor)

  • 2

  • 3

  • 4

  • 2

  • 3

🎫 Gerenciar Chamados

  • 4
    "Em limpeza agora": grade compacta com todos em andamento.

Supervisora

Supervisão e conferência de limpeza. Aprova ou reprova apartamentos após a limpeza.

🗺 Mapa 📋 Kanban 🎫 Chamados 👥 Equipe 📌 Minha Fila
📌 Conferência pela Minha Fila (fluxo principal)
A Minha Fila é a tela principal da supervisora. O sistema abre nela automaticamente ao fazer login.
  • 1
    Na seção "Aguardando Conferência", veja todos os aptos prontos para vistoria.
  • 2
    Para aprovar: clique Aprovar ✓. O apto muda para Limpo.
  • 3
    Para reprovar: clique Reprovar ✗, informe o motivo e confirme. O apto volta para Reprovado.
  • 4
    Clique em ↺ Atualizar para ver novos aptos prontos.
🎫 Chamados de Governança
  • 1
    Acessa todos os chamados de Governança do hotel (atribuídos e disponíveis).
  • 2
    Pode abrir novos chamados e assumir chamados sem responsável.
  • 3
    Recebe notificação em tempo real quando um novo chamado de governança é aberto no hotel.

Camareira

Execução da limpeza dos apartamentos. Interface simplificada para uso em celular ou tablet.

📱 App Camareira 🗺 Mapa 🎫 Chamados (Governança) 📌 Minha Fila
🧹 Meus Aptos Atualizado
A tela Meus Aptos exibe todos os apartamentos do hotel com ícone e status visual. Ideal para uso no celular durante o turno.
  • 1
    Clique em Meus Aptos no menu. Os apartamentos são exibidos em cards com ícone, número, andar, tipo e status.
  • 2
    Quando um gestor ou supervisora atribuir um apartamento diretamente a você, o card exibe o badge 👤 Atribuído a mim com borda azul — indicando que aquele apto é sua responsabilidade prioritária.
  • 3
    Use o filtro 👤 Meus atribuídos (N) para ver apenas os apartamentos atribuídos a você. O número entre parênteses indica a quantidade. Clique novamente para voltar a "Todos".
  • 4
    Use os demais botões de filtro por status (Sujo, Limpando, Pausado…) para exibir apenas um status específico. Os botões aparecem somente para status presentes no hotel no momento.
  • 5
    Toque em um card de apartamento Sujo para abrir o checklist e iniciar a limpeza.
📌 Minha Fila (tela principal) Atualizado
Ao fazer login, a camareira é direcionada automaticamente para Minha Fila.
  • 1
    A fila é ordenada por urgência: Reprovados → Pausados → Em limpeza → Sujos.
  • 2
    Dentro de cada grupo, apartamentos atribuídos diretamente a você aparecem primeiro, com um banner azul 📌 Atribuído a mim — prioridade de limpeza e fundo destacado. Trate-os antes dos demais.
  • 3
    Para iniciar: clique em ▶ Iniciar limpeza. O checklist de limpeza abre — o status não muda ainda.
  • 4
    No checklist, três opções disponíveis: Cancelar (mantém status atual), ⏸ Pausar (status → Pausado), ou ✓ Concluir limpeza (após preencher ≥ 80% dos itens → status → Em andamento).
  • 5
    O checklist não fecha ao clicar fora — é obrigatório pressionar um dos três botões.
  • 6
    Com status Em andamento: clique em 🔍 Enviar conf. para enviar à conferência. Status → Conferência. Aguarde a supervisora.
  • 7
    Se Reprovado: leia o motivo da reprovação no card e inicie o retrabalho com ▶ Iniciar limpeza.
Fluxo de status (camareira): Sujo → (checklist concluído) → Em andamento → (enviar conf.) → Ag. Conferência → (supervisora aprova) → Limpo
🎫 Chamados Atualizado
  • 1
    Clique em Chamados. A camareira vê todos os chamados do hotel (Governança + Manutenção).
  • 2
    Chamados de Governança: pode assumir, atualizar status, comentar e criar novos chamados.
  • 3
    Chamados de Manutenção: aparecem marcados com 👁 Somente leitura. Nenhuma ação disponível — apenas consulta.
  • 4
    Na aba Disponíveis, veja chamados de governança sem responsável. Clique em ✋ Assumir chamado para se tornar responsável.
  • 5
    Na aba Meus, veja apenas chamados atribuídos a você.
  • 6
    Recebe notificação modal quando um novo chamado de Governança é aberto no hotel — título "Novo Chamado Aberto". Pressione ✓ OK para confirmar o recebimento.
  • 7
    Recebe também notificação separada — título "Chamado Atribuído a Você" (ícone 📋) — quando um gestor ou supervisora atribui um chamado diretamente a você. Pressione ✓ OK para confirmar. O registro é salvo no histórico do chamado.
  • 8
    O badge numérico no menu Chamados mostra: chamados atribuídos a você + chamados de governança disponíveis no hotel.
  • 9
    Para abrir um chamado: + Novo Chamado → preencha tipo de chamado, apartamento, prioridade e descrição. O modal não fecha ao clicar fora.

🗺 Módulo: Mapa de Apartamentos

Disponível para: todos os perfis

Navegar no Mapa
  • 1
    Os apartamentos são exibidos agrupados por andar. Cada card exibe número, tipo, ícone de status e badges adicionais.
  • 2
    Use a barra de busca para localizar um apartamento pelo número.
  • 3
    Use os botões de filtro rápido (Todos / Sujo / Limpando / Conferência / Reprovado…) para filtrar por status.
  • 4

Alterar Status de um Apartamento
Disponível para todos os perfis. Qualquer usuário pode alterar o status operacional de um apartamento — a diferença é que apenas Admin pode editar dados cadastrais (número, andar, tipo).
  • 1
    Pelo card: clique no apartamento → na seção "Alterar Status", clique no novo status desejado.
  • 2
    Pelo cabeçalho: clique em Alterar Status → selecione o apartamento pelo número, escolha o novo status e confirme.
  • 3
    O mapa atualiza imediatamente e o histórico de status é registrado automaticamente.
Atribuir Camareira pelo Mapa Novo
Disponível para: Admin Global, Admin Hotel, Gestor, Supervisora. Camareira e Manutenção não visualizam esta opção.
  • 1
    Clique no card do apartamento para abrir o painel de detalhes.
  • 2
    Na seção 👤 Atribuir Camareira (parte inferior do painel), selecione a camareira desejada na lista suspensa.
  • 3
    A atribuição é salva automaticamente ao selecionar — não é necessário clicar em "Salvar". O nome da camareira atualiza no painel e no card do mapa.
  • 4
    Para remover a atribuição, selecione — Não atribuída na lista.
Admin: Vê também os botões ⊕ Cadastrar Apto e ⚡ Gerar em lote no cabeçalho. Esses botões não aparecem para outros perfis.
📋 Módulo: Kanban de Limpeza

Disponível para: Admin Global, Admin Hotel, Gestor, Supervisora

  • 1
    Os apartamentos são exibidos em colunas por status de limpeza: Aberto · Em análise · Em andamento · Pausado · Resolvido · Reaberto · Cancelado.
  • 2
    Use os filtros de Andar e Camareira para refinar a visualização.
  • 3
    Clique em qualquer card para ver detalhes e alterar o status diretamente.
Mapa vs. Kanban: O Mapa organiza por andar (visão espacial). O Kanban organiza por status de limpeza (visão de fluxo). Use o Kanban para acompanhar o andamento operacional.

Numeração Automática Novo

Todo chamado recebe um código único gerado automaticamente pelo banco de dados no momento da criação:

DepartamentoCódigo geradoExemplo
🧹 GovernançaGOV-XXXXXXGOV-000001
🔧 ManutençãoMAN-XXXXXXMAN-000001
O código aparece na listagem e no detalhe do chamado. Chamados criados antes desta funcionalidade exibem — no lugar do código.
Criar um Chamado
  • 1
    Clique em + Novo Chamado. O modal abre e não fecha ao clicar fora — use os botões Cancelar ou Abrir chamado.
  • 2
    Selecione o Departamento: Governança ou Manutenção. Os tipos de chamado disponíveis mudam conforme o departamento.
  • 3
    Preencha: Apartamento (opcional), Tipo de Chamado (obrigatório), Prioridade, Solicitante (lista carregada do banco — configurável em Config → Tipos de Solicitante), Responsável (opcional), Prazo, Descrição e Nome do Hóspede. Os campos Descrição e Nome do Hóspede são zerados automaticamente a cada abertura do modal — as informações já foram salvas no chamado anterior.
  • 4
    Se o apartamento selecionado já tiver chamados abertos, aparece um aviso: "⚠️ Já existe X chamado(s) aberto(s) para este apartamento." O aviso é informativo — a criação não é bloqueada.
  • 5
    Clique Abrir chamado. O código GOV-XXXXXX ou MAN-XXXXXX é gerado automaticamente. A equipe do hotel recebe notificação em tempo real.
Chamado sem responsável: Se o campo "Atribuir para" for deixado em branco, o chamado fica 📋 Disponível para toda a equipe do departamento no hotel.
Assumir um Chamado Disponível Novo
  • 1
    Na aba Disponíveis, clique no card do chamado.
  • 2
    No topo do detalhe, aparece o bloco amarelo: "Chamado sem responsável. Clique para assumir o atendimento."
  • 3
    Clique em ✋ Assumir chamado. Você se torna o responsável e o status avança de Aberto → Em andamento.
  • 4
    A ação é registrada no histórico: "Chamado assumido por [seu nome]."
  • 5
    O chamado sai da aba Disponíveis e passa para a aba Meus.

IndicadorComo é calculado
Produtividade hoje (%) Aptos que chegaram ao status Limpo hoje ÷ Aptos que saíram do status Sujo hoje × 100
Exemplo: 6 aptos limpos ÷ 8 aptos que iniciaram = 75%
Aptos/camareira hoje Aptos que chegaram ao status Limpo hoje ÷ Camareiras ativas na equipe

Ambos os indicadores são calculados com base no histórico de status do dia (meia-noite até agora) e resetam automaticamente a cada novo dia.

📱 Uso em Celular Atualizado
O GovHotel é totalmente responsivo. Em smartphones e tablets, a interface se adapta automaticamente para telas menores.
  • 1
    Toque no botão ☰ no canto superior esquerdo para abrir o menu lateral.
  • 2
    O menu lateral é rolável. Todos os itens de navegação ficam disponíveis na lista, incluindo 🔑 Alterar senha e ↪ Sair do sistema ao final do menu.
  • 3
    Toque em qualquer área fora do menu ou em um item de menu para fechar o painel lateral.
  • 4
    Para alterar a senha no celular: abra o menu → role até o final → toque em 🔑 Alterar senha.
  • 5
    Para sair do sistema no celular: abra o menu → role até o final → toque em ↪ Sair do sistema.
Recomendação: Camareiras e técnicos de manutenção normalmente utilizam o sistema pelo celular. Certifique-se de que a tela do dispositivo não adormeça durante o turno para receber notificações em tempo real.
❓ Dúvidas Frequentes

Esqueci minha senha, o que faço?

Entre em contato com o administrador do sistema ou do hotel. Ele pode redefinir sua senha pelo módulo Usuários.

Como alterar minha própria senha?

No desktop: clique em 🔑 Alterar senha no rodapé do menu lateral. No celular: abra o menu ☰ → role até o final → toque em 🔑 Alterar senha. Informe a nova senha (mínimo 6 caracteres) e confirme.

O sistema não atualizou automaticamente. O que fazer?

O GovHotel usa Supabase Realtime para atualizações automáticas via WebSocket. Se a conexão cair (ex.: troca de rede, tela bloqueada por muito tempo), recarregue a página para restabelecer a conexão. Use o botão ↺ Atualizar disponível em cada módulo para forçar o recarregamento.

O apartamento não aparece no mapa. O que pode ser?

Verifique se o apartamento está ativo no Cadastro de Aptos. Apartamentos inativos não aparecem no mapa. Apenas Admin pode reativar um apartamento.

Iniciei uma limpeza por engano, como desfazer?

O gestor, supervisora ou admin pode alterar o status do apartamento manualmente: Mapa → clique no apto → Forçar Status → escolha o status correto (ex.: Sujo).

Não consigo ver chamados na aba "Disponíveis". Por quê?

A aba Disponíveis mostra chamados sem responsável com status Aberto, Em análise ou Reaberto do departamento correto para o seu perfil. Se estiver vazia, não há chamados nessa condição no momento.

Posso reabrir um chamado já resolvido?

Sim. Abra o detalhe do chamado e clique em → Reaberto. O status volta para Reaberto e o histórico registra quem reabriu e quando.

O número de chamados no menu não bate com a listagem. O que pode ser?

O badge do menu conta todos os chamados abertos do hotel (status: Aberto + Em análise + Reaberto) de todos os departamentos, igual ao que a aba Abertos da listagem exibe. Não filtra por departamento — camareira e manutenção veem o mesmo total. Se o número ainda divergir, recarregue a página para sincronizar o cache.

Gestor consegue editar dados cadastrais de um apartamento?

Não. O gestor pode alterar o status operacional (ex.: Sujo → Limpando) mas não pode editar dados cadastrais como número, andar ou tipo. Essas alterações são exclusivas para Admin Hotel e Admin Global.

A equipe da página Equipe está vazia. O que verificar?

1. Verifique se o hotel correto está selecionado (Admin Global usa o seletor de hotel no topo). 2. Verifique se os usuários têm perfil camareira ou manutencao e estão ativos no módulo Usuários.
