📋 Visão Geral do Sistema

O Gov Estancorp é um sistema web de governança hoteleira que centraliza o controle de limpeza de apartamentos, chamados internos de governança e manutenção, indicadores operacionais em tempo real e gestão da equipe — tudo integrado ao banco de dados Supabase.

Acesso: Abra o sistema no navegador e faça login com seu usuário e senha fornecidos pelo administrador. O sistema abre automaticamente na tela correta para o seu perfil. Em celular, o menu lateral é acessado pelo botão ☰ no canto superior esquerdo.
PerfilFunçãoTelas disponíveis
Admin GlobalControla todos os hotéis da redeTodas as telas + Hotéis
Admin HotelControla um hotel específicoTodas exceto Hotéis
GestorGestão operacional do hotelDashboard, Mapa, Kanban, Chamados, Equipe, Relatórios, Minha Fila
SupervisoraSupervisão e conferência de limpezaMapa, Kanban, Chamados, Equipe, Minha Fila
CamareiraExecução da limpezaApp Camareira (Meus Aptos), Mapa, Chamados, Minha Fila
ManutençãoManutenção de apartamentosMapa, Chamados, Minha Fila (acesso somente leitura)
Dados sempre do banco: Todos os dados (apartamentos, equipe, chamados, categorias, motivos, checklists) são carregados diretamente do Supabase. Não há dados fictícios no sistema.

Chamados sem responsável — 📋 Disponível

Um chamado aberto sem responsável atribuído fica visível para toda a equipe do departamento correspondente no hotel. Qualquer membro autorizado pode clicar em ✋ Assumir chamado para se tornar o responsável.

Tipos de Chamado:

Os tipos de chamado são carregados do banco de dados por departamento (Governança ou Manutenção) e podem ser configurados pelo Admin em Config → Tipos de chamado.

🔔 Notificações em Tempo Real
Supabase Realtime + detecção automática: O sistema usa WebSocket para notificar a equipe em tempo real e também detecta novos chamados a cada atualização do cache, garantindo que nenhuma notificação seja perdida mesmo em caso de instabilidade de conexão.
  • 1
    Quando um novo chamado é aberto, os destinatários do departamento correspondente recebem um aviso modal na tela com o número do chamado, departamento, tipo e prioridade. O aviso não fecha automaticamente — exige ação explícita.
  • 2
    O destinatário deve pressionar ✓ OK — Confirmar recebimento. Ao confirmar, o sistema registra no histórico do chamado: "Recebimento confirmado por [nome] em [data/hora]".
  • 3
    Se chegarem múltiplos chamados enquanto o aviso está aberto, eles são enfileirados — ao confirmar um, o próximo aparece automaticamente.
  • 4
    A lista de chamados e o badge numérico do menu atualizam automaticamente em paralelo, sem ação do usuário.
  • 5
    Camareira recebe somente notificações de chamados de Governança. Manutenção recebe somente de Manutenção. Demais perfis não recebem aviso modal, mas o badge do menu é atualizado para todos.
  • 6
    Quem abriu o chamado não recebe notificação de si mesmo.
  • 7
    A conexão é encerrada automaticamente ao fazer logout.

Administrador Global

Controla todos os hotéis da rede. Acesso irrestrito a todas as funcionalidades.

🏨 Hotéis 👤 Usuários 📊 Dashboard 🗺 Mapa 📋 Kanban 🎫 Chamados 👥 Equipe 🛏 Cadastro Aptos 📈 Relatórios ⚙ Config 📌 Minha Fila
🏨 Gerenciar Hotéis
  • 1
    No menu lateral, clique em Hotéis.
  • 2
    Para adicionar: clique em + Novo Hotel, preencha nome, cidade, CNPJ e clique Salvar.
  • 3
    Para editar: clique no nome do hotel na lista e altere os dados.
  • 4
    Para desativar: toggle "Ativo" no cadastro. O hotel desaparece dos filtros, mas os dados são preservados.
👤 Gerenciar Usuários
  • 1
    No menu lateral, clique em Usuários.
  • 2
    Para criar: clique em + Novo Usuário, preencha nome, login (e-mail), perfil, hotel vinculado e senha temporária.
  • 3
    Marque "Forçar troca de senha no primeiro acesso" — o usuário definirá nova senha ao entrar.
  • 4
    Para desativar: clique no usuário → toggle "Ativo" → Salvar. Acesso é revogado imediatamente.
  • 5
    Para redefinir senha: clique no usuário → Redefinir senha → informe a nova senha temporária.
  • 6
    Somente o Admin Global pode criar e ver usuários com perfil Admin Global.
Isolamento por hotel: Cada usuário só enxerga dados do hotel ao qual está vinculado. O Admin Global enxerga todos os hotéis e pode selecionar o hotel desejado em cada módulo.
🛏 Cadastro de Apartamentos
Exclusivo para Admin: Apenas Admin Global e Admin Hotel podem cadastrar, editar e excluir apartamentos.
  • 1
    Clique em Cadastro Apto no menu lateral.
  • 2
    Para adicionar: clique em + Novo Apartamento, preencha número, andar, tipo e categoria, depois clique Salvar.
  • 3
    Para gerar em lote: clique em ⚡ Gerar em lote e configure o padrão de numeração e andar.
  • 4
    Para editar: clique no apartamento na tabela e altere os dados cadastrais.
  • 5
    Para excluir: clique no ícone de exclusão no card do apartamento e confirme.
⚙ Configurações do Sistema

O módulo Config centraliza todas as listas e parâmetros usados no sistema. Veja a seção completa em Módulo: Configurações.

Supervisora / Governanta

Supervisão e conferência de limpeza. Aprova ou reprova apartamentos após a limpeza.

🗺 Mapa 📋 Kanban 🎫 Chamados 👥 Equipe 📌 Minha Fila
📌 Conferência pela Minha Fila (fluxo principal)
A Minha Fila é a tela principal da supervisora. O sistema abre nela automaticamente ao fazer login.
  • 1
    Na seção "Aguardando Conferência", veja todos os aptos prontos para vistoria, com nome da camareira que realizou a limpeza.
  • 2
    Para aprovar: clique ✅ Aprovar limpeza. O apto muda para Limpo.
  • 3
    Para reprovar: clique ❌ Reprovar, selecione o motivo da lista (configurável em Config → Motivos de Reprovação), adicione uma observação opcional e confirme. O apto volta para Reprovado e o motivo fica registrado para a camareira.
  • 4
    O checklist de conferência (avaliado antes de aprovar/reprovar) é configurado em Config → Checklist de Conferência. Os itens representam os critérios que a supervisora, governanta ou gestor verificam na vistoria.
  • 5
    Clique em ↺ Atualizar para ver novos aptos prontos.
🎫 Chamados de Governança
  • 1
    Acessa todos os chamados de Governança do hotel (atribuídos e disponíveis).
  • 2
    Pode abrir novos chamados e assumir chamados sem responsável.
  • 3
    Recebe notificação em tempo real quando um novo chamado de governança é aberto no hotel.

Camareira

Execução da limpeza dos apartamentos. Interface simplificada para uso em celular ou tablet.

📱 Meus Aptos 🗺 Mapa 🎫 Chamados (Governança) 📌 Minha Fila
🧹 Meus Aptos Atualizado
A tela Meus Aptos exibe todos os apartamentos do hotel com ícone e status visual. O comportamento ao tocar em um card varia conforme o status do apartamento.
  • 1
    Clique em Meus Aptos no menu. Os apartamentos são exibidos em cards com ícone, número, andar, tipo e status.
  • 2
    Quando um gestor ou supervisora atribuir um apartamento diretamente a você, o card exibe o badge 👤 Atribuído a mim com borda azul.
  • 3
    Clique em Ver detalhes em qualquer card para abrir o painel completo do apartamento: histórico, status, observações e ações disponíveis.
  • 4
    Use o filtro 👤 Meus atribuídos (N) para ver apenas os apartamentos atribuídos a você.
  • 5
    Use os botões de filtro por status (Sujo, Em andamento, Pausado…) para exibir apenas um status específico.
Comportamento ao tocar em cada status:
Status do aptoO que abreAções disponíveis
Sujo Botões de ação diretos ▶ Iniciar limpeza → apto vai para Em andamento e o checklist de limpeza abre imediatamente
Em andamento Tela de ação direta ⏸ Pausar · 🚫 Cancelar limpeza (volta ao sujo) · 🔍 Enviar conf. (reabre checklist se fechou sem concluir)
Pausado Tela de ação direta ▶ Retomar limpeza → apto vai para Em andamento e checklist abre → ao concluir → Conferência
Reprovado Tela com motivo da reprovação ▶ Re-limpar → apto vai para Em andamento e checklist abre → ao concluir → Conferência
Fluxo completo: Sujo → (iniciar) → Em andamento + checklist abre → (concluir checklist) → Ag. Conferência → (supervisora aprova com checklist) → Limpo
📌 Minha Fila (tela principal)
Ao fazer login, a camareira é direcionada automaticamente para Minha Fila.
  • 1
    A fila é ordenada por urgência: Reprovados → Pausados → Em andamento → Sujos.
  • 2
    Dentro de cada grupo, apartamentos atribuídos diretamente a você aparecem primeiro com banner azul 📌 Atribuído a mim.
  • 3
    Para iniciar: clique em ▶ Iniciar limpeza. O apto vai para Em andamento e o checklist de limpeza abre automaticamente. Marque os itens (mínimo 80%) enquanto limpa.
  • 4
    Ao concluir o checklist, clique em ✓ Concluir limpeza. As respostas são salvas e o apto vai direto para Ag. Conferência. Se precisar cancelar a limpeza, clique em 🚫 Cancelar limpeza — o apto retorna ao status de origem.
  • 5
    Aguarde a supervisora aprovar ou reprovar. Se Reprovado: veja o motivo, clique em ▶ Re-limpar → apto volta para Em andamento, checklist abre novamente e o ciclo recomeça.
⏸ Pausar e Cancelar Limpeza

Motivos de Pausa

Ao pausar uma limpeza, a camareira seleciona o motivo na lista suspensa (configurável em Config → Motivos de Pausa):

Hóspede no quarto Aguardando enxoval Falta de material Solicitação da supervisão Interrupção operacional Outro

Motivos de Cancelamento

Ao cancelar uma limpeza (iniciada por engano ou por decisão operacional), selecione o motivo (configurável em Config → Motivos de Cancelamento):

Iniciado por engano Apartamento errado Hóspede não permitiu acesso Redistribuição de equipe Prioridade alterada Serviço cancelado pela governança Outro

🗺 Módulo: Mapa de Apartamentos Atualizado

Disponível para: todos os perfis

Navegar e Filtrar
  • 1
    Os apartamentos são exibidos agrupados por andar. Cada card exibe número, tipo, ícone de status, badges e tempo no status atual (ex.: "há 25min", "há 1h30").
  • 2
    Use a barra de busca para localizar um apartamento pelo número.
  • 3

  • 4

  • 5

Alterar Status de um Apartamento Atualizado
Status sensíveis (Limpando, Pausado, Ag. Conferência, Limpo, Reprovado) exigem: perfil Gestor, Supervisora, Governanta, Admin Hotel ou Admin Global + motivo obrigatório informado + confirmação. A alteração manual desses status é registrada no histórico. Camareira e Manutenção não conseguem alterar status sensíveis pelo modal manual.
  • 1
    Pelo card: clique no apartamento → na seção "Alterar Status", clique no novo status desejado.
  • 2
    Pelo cabeçalho: clique em Alterar Status → selecione o apartamento pelo número, escolha o status e preencha o motivo se obrigatório.
  • 3
    O mapa atualiza imediatamente e o histórico de status é registrado automaticamente.
Atribuir Camareira pelo Mapa
Disponível para: Admin Global, Admin Hotel, Gestor, Supervisora.
  • 1
    Clique no card do apartamento para abrir o painel de detalhes.
  • 2
    Na seção 👤 Atribuir Camareira, selecione a camareira desejada na lista suspensa.
  • 3
    A atribuição é salva automaticamente ao selecionar. Para remover, selecione — Não atribuída.
Ações em Lote Novo
Disponível para: Admin Global, Admin Hotel, Gestor, Supervisora, Governanta. Camareira e Manutenção não têm acesso às ações em lote.
  • 1
    Clique em ☑ Selecionar no cabeçalho do mapa para entrar no modo de seleção múltipla.
  • 2
    Clique nos cards dos apartamentos desejados para selecioná-los (ficam destacados). Use Selecionar todos para marcar todos do filtro atual.
  • 3
    Com apartamentos selecionados, escolha a ação:
    • Alterar Status: altera todos os selecionados para o status escolhido
    • 👤 Atribuir em lote: abre modal para atribuir a mesma camareira a todos os selecionados e salva camareira_id no cadastro do apto
    • Marcar Prioritário: ativa o indicador de prioridade em todos os selecionados
    • Remover Prioridade: remove o indicador de prioridade
  • 4
    Confirme a ação no diálogo de confirmação. O mapa atualiza ao finalizar.
  • 5
    Clique em Cancelar seleção ou no botão ☑ Selecionar novamente para sair do modo de seleção múltipla.
Admin: Vê também os botões ⊕ Cadastrar Apto e ⚡ Gerar em lote no cabeçalho. Esses botões não aparecem para outros perfis.
📋 Módulo: Kanban de Limpeza

Disponível para: Admin Global, Admin Hotel, Gestor, Supervisora

  • 1
    Os apartamentos são exibidos em colunas por status de limpeza: Sujo · Em limpeza · Pausado · Ag. Conferência · Limpo · Reprovado.
  • 2
    Use os filtros de Andar e Camareira para refinar a visualização.
  • 3

Mapa vs. Kanban: O Mapa organiza por andar (visão espacial). O Kanban organiza por status de limpeza (visão de fluxo). Use o Kanban para acompanhar o andamento operacional geral.

⚙ Módulo: Configurações Atualizado

Disponível para: Admin Global, Admin Hotel

Atenção: Alterações nas configurações afetam todos os usuários do hotel imediatamente. Desativar um tipo ou motivo impede que ele seja usado em novos registros.

Todos os itens abaixo seguem o mesmo padrão de interface: + Adicionar para incluir, ✏ para editar, ⏸/▶ para ativar/inativar, 🗑 para excluir. Registros marcados com 🌐 são globais (visíveis em todos os hotéis); registros 🏨 são específicos do hotel e podem ser excluídos.

🎫 Tipos de Chamado

Gerencie os tipos disponíveis por departamento (Governança / Manutenção). Ao abrir um chamado, apenas os tipos ativos do departamento selecionado são exibidos.

👥 Tipos de Solicitante

Gerencie quem pode abrir um chamado (ex.: Recepção, Hóspede, Gerência, Supervisora). Solicitantes inativos não aparecem no formulário de abertura de chamado.

❌ Motivos de Reprovação

Gerencie os motivos disponíveis quando a supervisora reprova a limpeza de um apartamento. Os motivos aparecem como lista suspensa para a supervisora no modal de reprovação.

Exemplos padrão: Banheiro incompleto · Cama mal feita · Piso com resíduos · Itens faltando (amenities) · Lixeiras não esvaziadas

⏸ Motivos de Pausa Novo

Gerencie os motivos exibidos quando a camareira pausa uma limpeza em andamento. O motivo é obrigatório ao pausar.

Exemplos padrão: Hóspede no quarto · Aguardando enxoval · Falta de material · Solicitação da supervisão · Interrupção operacional · Outro

🚫 Motivos de Cancelamento Novo

Gerencie os motivos exibidos quando uma limpeza é cancelada (diferente de pausada — cancelamento encerra a limpeza e retorna o apto para Sujo). O motivo é obrigatório ao cancelar.

Exemplos padrão: Iniciado por engano · Apartamento errado · Hóspede não permitiu acesso · Redistribuição de equipe · Prioridade alterada · Serviço cancelado pela governança · Outro

✅ Checklist de Limpeza (Camareira)

Itens que a camareira preenche ao iniciar a limpeza — o checklist abre automaticamente quando ela clica em "Iniciar limpeza", "Retomar" ou "Re-limpar". A limpeza só é enviada para conferência com ≥ 80% dos itens marcados.

🔍 Checklist de Conferência (Supervisora) Novo

Itens avaliados pela supervisora, governanta ou gestor antes de aprovar ou reprovar a limpeza do apartamento. Configure os critérios de vistoria específicos do hotel. Os itens aparecem no modal de conferência ao aprovar ou reprovar.

Exemplos padrão: Banheiro aprovado · Enxoval correto · Amenities corretos · Piso aprovado · Odor adequado · Frigobar conferido · Padrão visual aprovado · Nenhuma pendência aparente

🏠 Tipos de Apartamento Atualizado

Gerencie os tipos de apto disponíveis no cadastro (ex.: Standard, Luxo, Suíte, Superior). Os tipos ativos aparecem nos seletores de filtro do Mapa, Kanban e no formulário de cadastro de apartamentos.

🗂 Categorias de Apartamento Novo

Gerencie as categorias de apto (ex.: Casal, Solteiro, Triplo, Família). Utilizadas no cadastro de apartamentos e nos filtros do Mapa.

🧹 Tipos de Limpeza Novo

Gerencie os tipos de limpeza disponíveis (ex.: Diária, Saída, Permanência, Profunda). Usados no checklist e no histórico de limpeza para categorizar o serviço realizado.

🏗 Andares Novo

Gerencie os andares disponíveis no hotel. Os andares configurados aparecem como opções nos filtros do Mapa e Kanban e no formulário de cadastro de apartamentos. Permite personalizar a nomenclatura (ex.: Térreo, 1º, Subsolo).

👤 Cargos da Equipe Novo

❓ Dúvidas Frequentes

Esqueci minha senha, o que faço?

Entre em contato com o administrador do sistema. Ele pode redefinir sua senha pelo módulo Usuários.

Como alterar minha própria senha?

No desktop: clique em 🔑 Alterar senha no rodapé do menu lateral. No celular: abra o menu ☰ → role até o final → toque em 🔑 Alterar senha. Informe a nova senha (mínimo 6 caracteres) e confirme.

O sistema não atualizou automaticamente. O que fazer?

O Gov Estancorp usa Supabase Realtime via WebSocket. Se a conexão cair (ex.: troca de rede, tela bloqueada), recarregue a página para restabelecer. Use o botão ↺ Atualizar disponível em cada módulo para forçar o recarregamento.

O apartamento não aparece no mapa. O que pode ser?

Verifique se o apartamento está ativo no Cadastro de Aptos. Apartamentos inativos não aparecem no mapa. Apenas Admin pode reativar.

Iniciei uma limpeza por engano, como desfazer?

O gestor, supervisora ou admin pode cancelar: na Minha Fila ou no Mapa, use Cancelar limpeza e informe o motivo "Iniciado por engano". O apto retorna para Sujo.

Como alterar manualmente um status sensível (Limpando, Limpo, Reprovado)?

Apenas perfis Gestor, Supervisora, Governanta, Admin Hotel e Admin Global podem alterar esses status manualmente. Pelo Mapa → Alterar Status → selecione o apto e o novo status → preencha o motivo obrigatório → confirme o aviso. O histórico registra a alteração manual.

Como usar ações em lote no Mapa?

Clique em ☑ Selecionar no cabeçalho do Mapa para entrar no modo de seleção múltipla. Clique nos cards desejados, escolha a ação (Alterar Status, Atribuir Camareira, Marcar Prioritário) e confirme. Ações em lote estão disponíveis apenas para Gestor, Supervisora e Admin — Camareira e Manutenção não têm acesso.

O filtro "Sem camareira" não mostra nada. Por quê?

O filtro 👤 Sem camareira exibe somente apartamentos sujos sem responsável atribuído. Se o resultado estiver vazio, todos os aptos sujos já têm camareira atribuída ou não há aptos sujos no momento.

Não consigo ver chamados na aba "Disponíveis". Por quê?

A aba Disponíveis mostra chamados sem responsável com status Aberto, Em análise ou Reaberto do departamento correto para o seu perfil. Se estiver vazia, não há chamados nessa condição no momento.

Posso reabrir um chamado já resolvido?

Sim. Abra o detalhe do chamado e clique em → Reaberto. O status volta para Reaberto e o histórico registra quem reabriu e quando.

Gestor consegue editar dados cadastrais de um apartamento?

Não. O gestor pode alterar o status operacional (ex.: Sujo → Limpando) mas não pode editar dados cadastrais como número, andar ou tipo. Essas alterações são exclusivas para Admin Hotel e Admin Global.

Como adicionar um novo motivo de pausa ou cancelamento?

Acesse Config no menu lateral → aba Motivos de Pausa ou Motivos de Cancelamento → clique em + Adicionar → informe o nome e salve. O novo motivo aparece imediatamente nas listas suspensas para todos os usuários do hotel.

A equipe da página Equipe está vazia. O que verificar?

1. Verifique se o hotel correto está selecionado (Admin Global usa o seletor de hotel no topo). 2. Verifique se os usuários têm perfil camareira ou manutencao e estão ativos no módulo Usuários.

A tela de Minha Fila da manutenção está diferente das outras. É correto?

Sim. Para o perfil Manutenção, a Minha Fila exibe apenas um painel de boas-vindas com o total de aptos em manutenção e um atalho para Chamados — que é onde fica o fluxo de trabalho completo (assumir, andamento, conclusão).
